Introduction

What is DataUA?

DataUA (Data University Assistant) is a self-contained, browser-based application that transforms raw Excel or CSV grade sheets into formatted, printable assessment reports — with no installation required and no data ever leaving your device.

It is designed for instructors, TAs, and coordinators who need to process student assessment data quickly, apply a grading formula, visualize performance, and export the results — and is developed by Dataum, a startup focused on building customized, feature-rich applications for universities, factories, and companies.

🔒
Privacy First

All data is processed entirely in your browser. Nothing is sent to any server. You can even use this application offline once the page has loaded.

📋
What you need before starting

An Excel (.xlsx or .xls) or CSV file containing your student data. At minimum you need a column with student names and at least one column with numeric scores or grades.

01

Step One

Upload Your Data File

📁 Upload

Uploading an Excel or CSV File

On the first screen you will see a large upload area with a file picker. Click it to browse your computer, or simply drag and drop your file onto it.

  • Supported formats: .xlsx, .xls, and .csv
  • After selecting a file, the application will read it immediately — no "submit" button needed
  • Once the file is parsed, you are automatically taken to Step 2
⚠️
File format tips

If your Excel file has multiple sheets, only the first sheet is read. Make sure your data is on the first sheet before uploading.

💡
Don't have a file yet?

You can export any spreadsheet from Google Sheets as .xlsx or .csv via File → Download. For Moodle grade books, use the "Export → Excel spreadsheet" option.

02

Step Two

Select the Header Row

🗂️ Headers

Click the Row That Contains Your Column Names

The tool displays all rows from your file in a table. Your job here is to tell it which row holds the column headers (e.g., "Name", "Score", "Section").

  • Click anywhere on a row to highlight and select it as the header row
  • The selected row will be highlighted in blue
  • Everything above the selected row is ignored (useful if your file has a title or logo rows at the top)
  • Once selected, the app automatically advances to Step 3
ℹ️
Why this step exists

Many real-world spreadsheets have a title, institution name, or date in the first few rows before the actual data. This step lets you skip those rows and pick the correct header row no matter where it appears.

03

Step Three

Edit Your Data

✏️ Edit

Fix Errors Before Processing

The tool now shows your data in an editable table. You can correct typos, missing values, or wrong scores before the report is generated.

  • Click any cell to open a small editing dialog for that cell
  • Type the new value and click Save
  • You can also click column headers to sort the data
  • When you are satisfied, click the green "Done Editing" button at the top right
  • Clicking Done Editing advances you to Step 4 automatically
💡
It's okay to skip editing

If your data is already clean and correct, just click "Done Editing" immediately to proceed. This step only exists for convenience — it's not mandatory to change anything.

04

Step Four

Choose Display Columns

🗃️ Columns

Select What Appears in the Final Results Table

Your spreadsheet might have many columns (Student ID, Phone, Section, etc.). Here you decide which ones should appear in the final report — only the columns you care about.

  • The left panel lists all available columns from your file
  • Tick the checkbox next to each column you want to include
  • Selected columns appear in the right panel as draggable cards
  • Drag to reorder the right panel cards — the order here becomes the column order in the report
  • Use the search box to quickly find columns if you have many
  • Click "Select All" to include every column at once
  • Once at least one column is selected, the "Continue to Configuration" button activates — click it to proceed
ℹ️
The calculated score is always added automatically

You don't need to select the score column here. The tool will always append a "Score" and "Status" column at the end of your chosen display columns after calculation.

05

Step Five

Configure Column Roles & Grading Mode

This is the most important step. You tell the app two things: (1) what each column in your data means, and (2) how scores should be calculated.

⚙️ Configure

Part A — Column Mapping (Assign Roles to Columns)

The right side of the screen shows your data table. Click any column header to open a popup where you assign that column a role:

  • Name — The column containing student names (required)
  • Add (+) — A numeric column to be added to the score (e.g., quiz marks, attendance points)
  • Subtract (−) — A numeric column to be subtracted (e.g., penalty deductions)
  • Ignore — A column that exists in the data but should not affect the score at all
  • You can also set a Weight (multiplier) and an Offset (fixed bonus/penalty) per column

The left panel (Configuration Panel) shows a live summary of all assigned column roles. The "Generate Report" button activates once at least one Name column and one Add column have been assigned.

📊 Grading

Part B — Grading Mode

Below the column mapping section in the left panel, choose a grading mode. Each mode changes how the final score is interpreted and what badges/labels appear on students.

After configuring both parts, click the large "GENERATE REPORT" button. The app will calculate all scores and take you to Step 6 — Results.

⚠️
Required: At least one "Name" and one "Add" column

The "Generate Report" button stays disabled until you assign the Name role to one column and the Add role to at least one numeric column.

06

Step Six

Results, Charts & Export

📊 Results

Reading the Results Screen

The results page shows four summary statistics at the top: Total Students, Average Score, Highest Score, and Lowest Score. Below that you'll find:

  • Performance Chart — A visual chart of scores (bar, pie, line, radar, and more). Use the chart type tabs to switch views.
  • Multi-Chart Mode — Toggle this on to display multiple chart types simultaneously side by side.
  • Results Table — The ranked list of all students with their calculated scores, status badges, and any extra display columns you selected in Step 4.
  • Analytics Panel — Deeper statistical breakdowns: grade distribution, pass/fail counts, score bands, and percentile analysis.
  • Ranking View — Medal-style cards for top-ranked students (Gold, Silver, Bronze for the top 3).
💾 Export

Exporting & Saving Your Report

The toolbar at the top of the Results page has five action buttons:

💡
Loading Previously Saved Results

At any time, click the "Load Saved" button in the top navigation bar to restore a previously saved assessment without re-uploading any files.

⚠️
Starting a New Assessment

To process a completely different file, click "New Assessment" in the top navigation bar. This reloads the page and clears all current data.

Reference

All Grading Modes Explained

The tool offers 17 grading modes. Choose the one that matches your assessment context.

✓
Pass / Fail
Students are labelled Pass or Fail based on a configurable minimum score threshold.
🎭
Behavioral
Evaluates scores on behavioral/soft-skills criteria with descriptive labels.
⚖️
Weighted
Each score column can have a different weight. The final score is a weighted average.
💡
Creative
Designed for creative projects; applies bonus multipliers for exceptional scores.
🏗️
Project
Treats columns as project milestones; milestone weights are configurable.
📊
Percentile
Ranks students by percentile position within the class rather than raw score.
🔤
Letter Grade
Converts numeric scores to standard letter grades (A, B, C, D, F).
⚙️
Custom
Define your own score bands and labels for maximum flexibility.
📈
GPA
Converts scores to GPA points on a 4.0 scale using standard mapping.
🎯
Rubric
Multi-criteria rubric scoring; each column is a rubric dimension.
📊
Z-Score
Standardises scores relative to the class mean and standard deviation.
🏆
Contest / Competition
Designed for competitions; students are awarded ranks and medals (Gold/Silver/Bronze).
⭐
Star Rating
Converts scores to a 1–5 star visual rating system.
🎯
Mastery
Categorises students as Mastered, Developing, or Beginning based on thresholds.
📐
Curve
Applies a statistical bell-curve adjustment to normalize class scores.
🏅
Olympic
Ranks students with Olympic-style scoring; ties handled by time/secondary criteria.
📊
Portfolio
Four weighted dimensions: Technical (default 40%), Creativity (30%), Presentation (20%), Documentation (10%).
🎲
Gamified
Assigns XP (experience points) and achievement levels in a game-style format.
💡
Portfolio Mode Tip

In Portfolio mode, a "Validate Weights" button appears. Make sure Technical + Creativity + Presentation + Documentation adds up to exactly 100% before generating the report.

Tips & Tricks

Keyboard Shortcuts & Power User Features

Ctrl + S Save current results to browser storage
Ctrl + P Print / open print dialog (results only)
Ctrl + E Export results to CSV

Theme & Appearance

The app includes four built-in themes: Light (default), Dark, DataUA Official (branded navy/orange theme), and High Contrast (accessibility). Click the floating button at the bottom-right of the screen to switch themes.

The sidebar panel (accessible via the tab button on the right edge) allows full color customization, layout density adjustments, and toggles for reduced motion and split-view results.

Save & Load Sessions

After generating results, use the Save button to store your assessment in the browser. Your saved assessments persist even after closing the tab (until browser data is cleared).

To reload a saved session without re-uploading your file, click "Load Saved" from the top navigation bar. A modal will appear listing all saved assessments with timestamps. You can load or delete each one.

Multi-Chart View

On the Results page, look for the "Multi-Chart Mode" toggle. When enabled, a panel appears letting you tick which chart types to display simultaneously. Up to 8 chart types can be shown in a 2-column grid layout.

Table View Modes

The results table can be switched between three views using the view-type buttons above it: Ranked (sorted by score), Alphabetical (sorted by name), and Custom (preserving original upload order).

Analytics Tab

Click the Analytics tab on the results page to see deeper breakdowns: grade distribution histograms, pass/fail ratios, score range analysis, and a percentile table. This data is also included in PDF exports.
