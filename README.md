# DataUA — Data University Assistant

**by Dataum**

A browser-based academic reporting tool that transforms Excel and CSV grade sheets into formatted, printable assessment reports — with no installation required and no data ever leaving your device.

🔗 **Live App:** [dataua.netlify.app](https://dataua.netlify.app)

## Features
- Upload Excel (.xlsx, .xls) or CSV files
- 6-step guided workflow from raw data to final report
- 17 grading modes (Pass/Fail, GPA, Letter Grade, Z-Score, and more)
- Interactive charts (Bar, Pie, Radar, Scatter, and more)
- Export to PDF, Excel, CSV, or Print
- 4 built-in themes including Dark mode
- Runs 100% locally — no server, no login, no data upload
